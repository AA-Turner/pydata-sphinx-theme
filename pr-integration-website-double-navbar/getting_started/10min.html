10 Minutes to pandas¶

This is a short introduction to pandas, geared mainly for new users. You can see more complex recipes in the Cookbook.

Customarily, we import as follows:

In [9]: import numpy as np

In [10]: import pandas as pd

Object Creation¶

See the Data Structure Intro section.

Creating a Series by passing a list of values, letting pandas create a default integer index:

In [11]: s = pd.Series([1, 3, 5, np.nan, 6, 8])

In [12]: s

Creating a DataFrame by passing a NumPy array, with a datetime index and labeled columns:

In [13]: dates = pd.date_range('20130101', periods=6)

In [14]: dates

In [15]: df = pd.DataFrame(np.random.randn(6, 4), index=dates, columns=list('ABCD'))

In [16]: df

Creating a DataFrame by passing a dict of objects that can be converted to series-like.

In [17]: df2 = pd.DataFrame({'A': 1.,
   ....:                     'B': pd.Timestamp('20130102'),
   ....:                     'C': pd.Series(1, index=list(range(4)), dtype='float32'),
   ....:                     'D': np.array([3] * 4, dtype='int32'),
   ....:                     'E': pd.Categorical(["test", "train", "test", "train"]),
   ....:                     'F': 'foo'})
   ....: 

In [18]: df2

The columns of the resulting DataFrame have different dtypes.

In [19]: df2.dtypes

If you’re using IPython, tab completion for column names (as well as public attributes) is automatically enabled. Here’s a subset of the attributes that will be completed:

In [20]: df2.<TAB>  # noqa: E225, E999
df2.A                  df2.bool
df2.abs                df2.boxplot
df2.add                df2.C
df2.add_prefix         df2.clip
df2.add_suffix         df2.clip_lower
df2.align              df2.clip_upper
df2.all                df2.columns
df2.any                df2.combine

As you can see, the columns A, B, C, and D are automatically tab completed. E is there as well; the rest of the attributes have been truncated for brevity.

Viewing Data¶

See the Basics section.

Here is how to view the top and bottom rows of the frame:

In [21]: df.head()

In [22]: df.tail(3)

Display the index, columns:

In [23]: df.index

In [24]: df.columns

DataFrame.to_numpy() gives a NumPy representation of the underlying data. Note that this can be an expensive operation when your DataFrame has columns with different data types, which comes down to a fundamental difference between pandas and NumPy: NumPy arrays have one dtype for the entire array, while pandas DataFrames have one dtype per column. When you call DataFrame.to_numpy(), pandas will find the NumPy dtype that can hold all of the dtypes in the DataFrame. This may end up being object, which requires casting every value to a Python object.

For df, our DataFrame of all floating-point values, DataFrame.to_numpy() is fast and doesn’t require copying data.

In [25]: df.to_numpy()

For df2, the DataFrame with multiple dtypes, DataFrame.to_numpy() is relatively expensive.

In [26]: df2.to_numpy()

Note

DataFrame.to_numpy() does not include the index or column labels in the output.

describe() shows a quick statistic summary of your data:

In [27]: df.describe()

Transposing your data:

In [28]: df.T

Sorting by an axis:

In [29]: df.sort_index(axis=1, ascending=False)

Sorting by values:

In [30]: df.sort_values(by='B')

Selection¶

Note

While standard Python / Numpy expressions for selecting and setting are intuitive and come in handy for interactive work, for production code, we recommend the optimized pandas data access methods, .at, .iat, .loc and .iloc.

See the indexing documentation Indexing and Selecting Data and MultiIndex / Advanced Indexing.

Getting¶

Selecting a single column, which yields a Series, equivalent to df.A:

In [31]: df['A']

Selecting via [], which slices the rows.

In [32]: df[0:3]

In [33]: df['20130102':'20130104']

Selection by Label¶

See more in Selection by Label.

For getting a cross section using a label:

In [34]: df.loc[dates[0]]

Selecting on a multi-axis by label:

In [35]: df.loc[:, ['A', 'B']]

Showing label slicing, both endpoints are included:

In [36]: df.loc['20130102':'20130104', ['A', 'B']]

Reduction in the dimensions of the returned object:

In [37]: df.loc['20130102', ['A', 'B']]

For getting a scalar value:

In [38]: df.loc[dates[0], 'A']

For getting fast access to a scalar (equivalent to the prior method):

In [39]: df.at[dates[0], 'A']

Selection by Position¶

See more in Selection by Position.

Select via the position of the passed integers:

In [40]: df.iloc[3]

By integer slices, acting similar to numpy/python:

In [41]: df.iloc[3:5, 0:2]

By lists of integer position locations, similar to the numpy/python style:

In [42]: df.iloc[[1, 2, 4], [0, 2]]

For slicing rows explicitly:

In [43]: df.iloc[1:3, :]

For slicing columns explicitly:

In [44]: df.iloc[:, 1:3]

For getting a value explicitly:

In [45]: df.iloc[1, 1]

For getting fast access to a scalar (equivalent to the prior method):

In [46]: df.iat[1, 1]

Boolean Indexing¶

Using a single column’s values to select data.

In [47]: df[df.A > 0]

Selecting values from a DataFrame where a boolean condition is met.

In [48]: df[df > 0]

Using the isin() method for filtering:

In [49]: df2 = df.copy()

In [50]: df2['E'] = ['one', 'one', 'two', 'three', 'four', 'three']

In [51]: df2

In [52]: df2[df2['E'].isin(['two', 'four'])]

Setting¶

Setting a new column automatically aligns the data by the indexes.

In [53]: s1 = pd.Series([1, 2, 3, 4, 5, 6], index=pd.date_range('20130102', periods=6))

In [54]: s1

In [55]: df['F'] = s1

Setting values by label:

In [56]: df.at[dates[0], 'A'] = 0

Setting values by position:

In [57]: df.iat[0, 1] = 0

Setting by assigning with a NumPy array:

In [58]: df.loc[:, 'D'] = np.array([5] * len(df))

The result of the prior setting operations.

In [59]: df

A where operation with setting.

In [60]: df2 = df.copy()

In [61]: df2[df2 > 0] = -df2

In [62]: df2

Missing Data¶

pandas primarily uses the value np.nan to represent missing data. It is by default not included in computations. See the Missing Data section.

Reindexing allows you to change/add/delete the index on a specified axis. This returns a copy of the data.

In [63]: df1 = df.reindex(index=dates[0:4], columns=list(df.columns) + ['E'])

In [64]: df1.loc[dates[0]:dates[1], 'E'] = 1

In [65]: df1

To drop any rows that have missing data.

In [66]: df1.dropna(how='any')

Filling missing data.

In [67]: df1.fillna(value=5)

To get the boolean mask where values are nan.

In [68]: pd.isna(df1)

Stats¶

Operations in general exclude missing data.

Performing a descriptive statistic:

In [69]: df.mean()

Same operation on the other axis:

In [70]: df.mean(1)

Operating with objects that have different dimensionality and need alignment. In addition, pandas automatically broadcasts along the specified dimension.

In [71]: s = pd.Series([1, 3, 5, np.nan, 6, 8], index=dates).shift(2)

In [72]: s

In [73]: df.sub(s, axis='index')

Apply¶

Applying functions to the data:

In [74]: df.apply(np.cumsum)

In [75]: df.apply(lambda x: x.max() - x.min())

Histogramming¶

In [77]: s

In [78]: s.value_counts()

String Methods¶

Series is equipped with a set of string processing methods in the str attribute that make it easy to operate on each element of the array, as in the code snippet below. Note that pattern-matching in str generally uses regular expressions by default (and in some cases always uses them). See more at Vectorized String Methods.

In [79]: s = pd.Series(['A', 'B', 'C', 'Aaba', 'Baca', np.nan, 'CABA', 'dog', 'cat'])

In [80]: s.str.lower()

Concat¶

pandas provides various facilities for easily combining together Series and DataFrame objects with various kinds of set logic for the indexes and relational algebra functionality in the case of join / merge-type operations.

See the Merging section.

Concatenating pandas objects together with concat():

In [81]: df = pd.DataFrame(np.random.randn(10, 4))

In [82]: df

# break it into pieces
In [83]: pieces = [df[:3], df[3:7], df[7:]]

In [84]: pd.concat(pieces)

Join¶

SQL style merges. See the Database style joining section.

In [85]: left = pd.DataFrame({'key': ['foo', 'foo'], 'lval': [1, 2]})

In [86]: right = pd.DataFrame({'key': ['foo', 'foo'], 'rval': [4, 5]})

In [87]: left

In [88]: right

In [89]: pd.merge(left, right, on='key')

Another example that can be given is:

In [90]: left = pd.DataFrame({'key': ['foo', 'bar'], 'lval': [1, 2]})

In [91]: right = pd.DataFrame({'key': ['foo', 'bar'], 'rval': [4, 5]})

In [92]: left

In [93]: right

In [94]: pd.merge(left, right, on='key')

In [98]: df.append(s, ignore_index=True)

Grouping¶

By “group by” we are referring to a process involving one or more of the following steps:

  • Splitting the data into groups based on some criteria

  • Applying a function to each group independently

  • Combining the results into a data structure

See the Grouping section.

In [99]: df = pd.DataFrame({'A': ['foo', 'bar', 'foo', 'bar',
   ....:                          'foo', 'bar', 'foo', 'foo'],
   ....:                    'B': ['one', 'one', 'two', 'three',
   ....:                          'two', 'two', 'one', 'three'],
   ....:                    'C': np.random.randn(8),
   ....:                    'D': np.random.randn(8)})
   ....: 

In [100]: df

Grouping and then applying the sum() function to the resulting groups.

In [101]: df.groupby('A').sum()

Grouping by multiple columns forms a hierarchical index, and again we can apply the sum function.

In [102]: df.groupby(['A', 'B']).sum()

Stack¶

In [103]: tuples = list(zip(*[['bar', 'bar', 'baz', 'baz',
   .....:                      'foo', 'foo', 'qux', 'qux'],
   .....:                     ['one', 'two', 'one', 'two',
   .....:                      'one', 'two', 'one', 'two']]))
   .....: 

In [104]: index = pd.MultiIndex.from_tuples(tuples, names=['first', 'second'])

In [105]: df = pd.DataFrame(np.random.randn(8, 2), index=index, columns=['A', 'B'])

In [106]: df2 = df[:4]

In [107]: df2

The stack() method “compresses” a level in the DataFrame’s columns.

In [108]: stacked = df2.stack()

In [109]: stacked

With a “stacked” DataFrame or Series (having a MultiIndex as the index), the inverse operation of stack() is unstack(), which by default unstacks the last level:

In [110]: stacked.unstack()

In [111]: stacked.unstack(1)

In [112]: stacked.unstack(0)

Time Series¶

pandas has simple, powerful, and efficient functionality for performing resampling operations during frequency conversion (e.g., converting secondly data into 5-minutely data). This is extremely common in, but not limited to, financial applications. See the Time Series section.

In [116]: rng = pd.date_range('1/1/2012', periods=100, freq='S')

In [117]: ts = pd.Series(np.random.randint(0, 500, len(rng)), index=rng)

In [118]: ts.resample('5Min').sum()

Time zone representation:

In [119]: rng = pd.date_range('3/6/2012 00:00', periods=5, freq='D')

In [120]: ts = pd.Series(np.random.randn(len(rng)), rng)

In [121]: ts

In [122]: ts_utc = ts.tz_localize('UTC')

In [123]: ts_utc

Converting to another time zone:

In [124]: ts_utc.tz_convert('US/Eastern')

Converting between time span representations:

In [125]: rng = pd.date_range('1/1/2012', periods=5, freq='M')

In [126]: ts = pd.Series(np.random.randn(len(rng)), index=rng)

In [127]: ts

In [128]: ps = ts.to_period()

In [129]: ps

In [130]: ps.to_timestamp()

Converting between period and timestamp enables some convenient arithmetic functions to be used. In the following example, we convert a quarterly frequency with year ending in November to 9am of the end of the month following the quarter end:

In [131]: prng = pd.period_range('1990Q1', '2000Q4', freq='Q-NOV')

In [132]: ts = pd.Series(np.random.randn(len(prng)), prng)

In [133]: ts.index = (prng.asfreq('M', 'e') + 1).asfreq('H', 's') + 9

In [134]: ts.head()

Categoricals¶

pandas can include categorical data in a DataFrame. For full docs, see the categorical introduction and the API documentation.

In [135]: df = pd.DataFrame({"id": [1, 2, 3, 4, 5, 6],
   .....:                    "raw_grade": ['a', 'b', 'b', 'a', 'a', 'e']})
   .....:

Convert the raw grades to a categorical data type.

In [136]: df["grade"] = df["raw_grade"].astype("category")

In [137]: df["grade"]

Rename the categories to more meaningful names (assigning to Series.cat.categories is inplace!).

In [138]: df["grade"].cat.categories = ["very good", "good", "very bad"]

Reorder the categories and simultaneously add the missing categories (methods under Series .cat return a new Series by default).

In [139]: df["grade"] = df["grade"].cat.set_categories(["very bad", "bad", "medium",
   .....:                                               "good", "very good"])
   .....: 

In [140]: df["grade"]

Sorting is per order in the categories, not lexical order.

In [141]: df.sort_values(by="grade")

Grouping by a categorical column also shows empty categories.

In [142]: df.groupby("grade").size()

>>> if pd.Series([False, True, False]):
...     print("I was true")
Traceback
    ...
ValueError: The truth value of an array is ambiguous. Use a.empty, a.any() or a.all().

See Comparisons for an explanation and what to do.
