Cookbook¶

This is a repository for short and sweet examples and links for useful pandas recipes. We encourage users to add to this documentation.

Adding interesting links and/or inline examples to this section is a great First Pull Request.

Simplified, condensed, new-user friendly, in-line examples have been inserted where possible to augment the Stack-Overflow and GitHub links. Many of the links contain expanded information, above what the in-line examples offer.

Pandas (pd) and Numpy (np) are the only two abbreviated imported modules. The rest are kept explicitly imported for newer users.

These examples are written for Python 3. Minor tweaks might be necessary for earlier python versions.

Idioms¶

These are some neat pandas idioms

if-then/if-then-else on one column, and assignment to another one or more columns:

In [9]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ...:                    'BBB': [10, 20, 30, 40],
   ...:                    'CCC': [100, 50, -30, -50]})
   ...: 

In [10]: df

if-then…¶

An if-then on one column

In [11]: df.loc[df.AAA >= 5, 'BBB'] = -1

In [12]: df

An if-then with assignment to 2 columns:

In [13]: df.loc[df.AAA >= 5, ['BBB', 'CCC']] = 555

In [14]: df

Add another line with different logic, to do the -else

In [15]: df.loc[df.AAA < 5, ['BBB', 'CCC']] = 2000

In [16]: df

Or use pandas where after you’ve set up a mask

In [17]: df_mask = pd.DataFrame({'AAA': [True] * 4,
   ....:                         'BBB': [False] * 4,
   ....:                         'CCC': [True, False] * 2})
   ....: 

In [18]: df.where(df_mask, -1000)

if-then-else using numpy’s where()

In [19]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ....:                    'BBB': [10, 20, 30, 40],
   ....:                    'CCC': [100, 50, -30, -50]})
   ....: 

In [20]: df

In [21]: df['logic'] = np.where(df['AAA'] > 5, 'high', 'low')

In [22]: df

Building Criteria¶

Select with multi-column criteria

In [27]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ....:                    'BBB': [10, 20, 30, 40],
   ....:                    'CCC': [100, 50, -30, -50]})
   ....: 

In [28]: df

…and (without assignment returns a Series)

In [29]: df.loc[(df['BBB'] < 25) & (df['CCC'] >= -40), 'AAA']

…or (without assignment returns a Series)

In [30]: df.loc[(df['BBB'] > 25) | (df['CCC'] >= -40), 'AAA']

…or (with assignment modifies the DataFrame.)

In [31]: df.loc[(df['BBB'] > 25) | (df['CCC'] >= 75), 'AAA'] = 0.1

In [32]: df

Select rows with data closest to certain value using argsort

In [33]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ....:                    'BBB': [10, 20, 30, 40],
   ....:                    'CCC': [100, 50, -30, -50]})
   ....: 

In [34]: df

In [35]: aValue = 43.0

In [36]: df.loc[(df.CCC - aValue).abs().argsort()]

Dynamically reduce a list of criteria using a binary operators

In [37]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ....:                    'BBB': [10, 20, 30, 40],
   ....:                    'CCC': [100, 50, -30, -50]})
   ....: 

In [38]: df

In [39]: Crit1 = df.AAA <= 5.5

In [40]: Crit2 = df.BBB == 10.0

In [41]: Crit3 = df.CCC > -40.0

One could hard code:

In [42]: AllCrit = Crit1 & Crit2 & Crit3

…Or it can be done with a list of dynamically built criteria

In [43]: import functools

In [44]: CritList = [Crit1, Crit2, Crit3]

In [45]: AllCrit = functools.reduce(lambda x, y: x & y, CritList)

In [46]: df[AllCrit]

The indexing docs.

Using both row labels and value conditionals

In [47]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ....:                    'BBB': [10, 20, 30, 40],
   ....:                    'CCC': [100, 50, -30, -50]})
   ....: 

In [48]: df

Use loc for label-oriented slicing and iloc positional slicing

In [50]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ....:                    'BBB': [10, 20, 30, 40],
   ....:                    'CCC': [100, 50, -30, -50]},
   ....:                   index=['foo', 'bar', 'boo', 'kar'])
   ....:

There are 2 explicit slicing methods, with a third general case

  1. Positional-oriented (Python slicing style : exclusive of end)
  2. Label-oriented (Non-Python slicing style : inclusive of end)
  3. General (Either slicing style : depends on if the slice contains labels or positions)
In [51]: df.loc['bar':'kar']  # Label

# Generic
In [52]: df.iloc[0:3]

In [53]: df.loc['bar':'kar']

Ambiguity arises when an index consists of integers with a non-zero start or non-unit increment.

In [54]: data = {'AAA': [4, 5, 6, 7],
   ....:         'BBB': [10, 20, 30, 40],
   ....:         'CCC': [100, 50, -30, -50]}
   ....: 

In [55]: df2 = pd.DataFrame(data=data, index=[1, 2, 3, 4])  # Note index starts at 1.

In [56]: df2.iloc[1:3]  # Position-oriented

In [57]: df2.loc[1:3]  # Label-oriented

Using inverse operator (~) to take the complement of a mask

In [58]: df = pd.DataFrame({'AAA': [4, 5, 6, 7],
   ....:                    'BBB': [10, 20, 30, 40],
   ....:                    'CCC': [100, 50, -30, -50]})
   ....: 

In [59]: df

In [60]: df[~((df.AAA <= 6) & (df.index.isin([0, 2, 4])))]

Binary Files¶

pandas readily accepts NumPy record arrays, if you need to read in a binary file consisting of an array of C structs. For example, given this C program in a file called main.c compiled with gcc main.c -std=gnu99 on a 64-bit machine,

#include <stdio.h>
#include <stdint.h>

typedef struct _Data
{
    int32_t count;
    double avg;
    float scale;
} Data;

int main(int argc, const char *argv[])
{
    size_t n = 10;
    Data d[n];

    for (int i = 0; i < n; ++i)
    {
        d[i].count = i;
        d[i].avg = i + 1.0;
        d[i].scale = (float) i + 2.0f;
    }

    FILE *file = fopen("binary.dat", "wb");
    fwrite(&d, sizeof(Data), n, file);
    fclose(file);

    return 0;
}

the following Python code will read the binary file 'binary.dat' into a pandas DataFrame, where each element of the struct corresponds to a column in the frame:

names = 'count', 'avg', 'scale'

# note that the offsets are larger than the size of the type because of
# struct padding
offsets = 0, 8, 16
formats = 'i4', 'f8', 'f4'
dt = np.dtype({'names': names, 'offsets': offsets, 'formats': formats},
              align=True)
df = pd.DataFrame(np.fromfile('binary.dat', dt))

Note

The offsets of the structure elements may be different depending on the architecture of the machine on which the file was created. Using a raw binary file format like this for general data storage is not recommended, as it is not cross platform. We recommended either HDF5 or msgpack, both of which are supported by pandas’ IO facilities.
